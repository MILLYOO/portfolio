```JAVA 17``` ```SpringBoot 2.6.7``` ```Node 16.17.0``` ```Vue 2.7.10``` ```Nginx```

- 입학지원자, 관리자 화면 구현 및 개발
- PayPal, KG이니시스 API를 활용한 결제 시스템 구현
- i18n을 활용한 다국어 처리
- 스케줄러를 통한 외부 ERP 권한 연동 처리

#### 🔴 Issue
- 프론트엔드 i18n 적용 이후에도 Spring 서버가 반환하는 유효성 오류·안내 메시지가 한국어로 고정되어 영어 환경의 해외 지원자에게 그대로 노출
- 외부 ERP 권한 연동 스케줄러가 매 실행마다 전체 사용자를 재조회해 처리 대상이 불필요하게 많아지는 구조

#### 🟡 Solution
- Spring `MessageSource` + `Accept-Language` 헤더 기반으로 서버 응답 메시지도 언어별로 분기 처리
- 마지막 동기화 시점 이후 변경된 데이터만 조회하는 증분 처리 방식으로 스케줄러 로직 개선

#### 🟢 Result
- 서버 응답 메시지를 포함한 전 화면 다국어 지원 완성
- ERP 연동 스케줄 처리 대상 건수 감소로 수행 시간 단축
