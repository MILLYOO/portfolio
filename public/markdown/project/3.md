```JAVA 17``` ```SpringBoot 2.6.7``` ```Node 16.17.0``` ```Vue 2.7.10``` ```Nginx```

- Vue.js를 활용한 임용지원자 및 관리자 화면 구현 및 기능 개발
- NAVER CLOUD PLATFORM API를 이용한 이메일, 알림톡 화면 및 기능 구현
- OPEN API를 통한 논문, 도서 검색 연동 화면 및 기능 구현
- 각종 게시판 구현 및 권한별 버튼 구현

#### 🔴 Issue
- 임용 단계 전환 시 다수 지원자에게 알림톡·이메일을 동기로 발송하면서 API 응답 대기로 인한 처리 지연 발생
- 권한에 따른 버튼 제어가 프론트엔드에서만 이루어져 API를 직접 호출하면 권한 외 기능에 접근 가능한 구조

#### 🟡 Solution
- 발송 요청을 DB 발송 이력 테이블에 등록하고 스케줄러가 비동기로 처리, 실패 건은 재발송 대상으로 상태 관리
- 서버사이드에서 역할(Role) 기반 권한을 재검증하는 로직을 추가해 프론트-백엔드 이중 방어 구조 적용

#### 🟢 Result
- 알림 발송이 메인 처리 흐름에서 분리되어 단계 전환 응답 지연 해소 및 발송 안정성 확보
- API 레벨 권한 검증으로 프론트 우회를 통한 비인가 접근 차단
