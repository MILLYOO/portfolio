```JAVA 17``` ```SpringBoot 2.6.7``` ```Vue 2.7.10``` ```Nginx```

- SpringSecurity OAuth2를 활용한 소셜 로그인 및 연동 구현 (네이버, 카카오, 구글)
- Instagram Graph API를 활용한 피드 실시간 연동 구현
- 권한별 게시판 및 기능 구현

#### 🔴 Issue
- Instagram Graph API 액세스 토큰이 60일마다 만료되어 피드 연동이 주기적으로 끊기는 문제 발생
- 소셜 로그인 Provider(네이버·카카오·구글)별 OAuth2 응답 구조가 달라 통합 처리 시 분기 로직 복잡도 증가

#### 🟡 Solution
- Long-lived 토큰 발급 후 만료 전 자동 갱신하는 스케줄러 구성으로 토큰 관리 자동화
- Spring Security OAuth2 공통 핸들러를 구성해 Provider별 응답을 단일 UserInfo 객체로 정규화

#### 🟢 Result
- Instagram 피드 연동 중단 없이 안정적인 서비스 운영
- 소셜 로그인 코드 중복 제거로 신규 Provider 추가 용이한 구조 확보
